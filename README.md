# ⏱️ Modern React Stopwatch

A clean and modern stopwatch built using **React** with a professional UI design.

This project demonstrates the use of **React Hooks** like `useState` and `useRef` to build a functional stopwatch with a simple and elegant user interface.

---

## 🚀 Features

- Start the stopwatch
- Stop the stopwatch
- Reset the stopwatch
- Real-time time tracking
- Modern and clean UI design
- Responsive layout
- Beginner-friendly code

---

## 🛠️ Technologies Used

- React
- JavaScript (ES6)
- HTML5
- CSS3
